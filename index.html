浏览器可用

GitHub Pages 上直接填写和生成 Word，不需要安装 Python，也不需要后端服务器。

本地隐私

草稿和历史版本存放在用户浏览器 localStorage，不会上传到 GitHub 或其它服务器。

两种使用方式

打开网页：
https://mi0uno.github.io/MeowResume/app.html
本地完整版：
pip install -r requirements.txt
python web_server.py
隐私说明： 在线版在浏览器本地运行，适合快速生成简历。需要嵌入真实证件照或使用更严格的 Python 版模板时，可以拉取仓库后本地运行。
